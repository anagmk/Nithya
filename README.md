# Nithya 📚

An educational presentation platform built for teachers.  
Teachers can display teaching materials as slides and attach  
questions — students can learn the content and attend tests.

## Features

- 📖 Present teaching materials in a clean slide format
- ❓ Add questions to each topic
- ✅ Students can learn and attend tests
- 🖥️ Runs entirely in the browser — no backend needed

## Tech stack

- HTML (frontend only)

## How to run

1. Clone the repo
   git clone https://github.com/anagmk/Nithya.git
2. Open index.html in your browser

## Use case

Designed for classroom use — teachers can prepare  
lessons with questions and share them with students  
as a simple, no-login learning tool.
